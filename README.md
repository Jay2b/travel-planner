# Travel Planner

Fullstack application for creating and managing travel plans. Users can add, edit, and delete trips, with data stored in a database and handled through a custom REST API.

The backend is deployed on Render free tier, so the first request may take up to minute to wake up.

## Features

- View all trips
- Add new trip
- Edit existing trip
- Delete trip
- Data persisted in SQLite database

## Tech Stack

Frontend:
- React (Vite)
- Fetch API
- CSS

Backend:
- Node.js
- Express
- SQLite
- REST API
